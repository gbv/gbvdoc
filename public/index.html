Über http://uri.gbv.de/document/ können Linked Data URIs von Dokumenten in Bibliothekskatalogen - insbesondere von Institutionen im GBV - abgefragt werden. Diese umschließen sowohl im Functional Requirements for Bibliographic Records (FRBR) beschriebene physische und digitale Medien, die Document Availability Information API (DAIA), als auch generelle Typen der Bibliographic Ontology (BIBO).
