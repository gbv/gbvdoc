At http://uri.gbv.de/document/ you find Linked Data URIs of single and collected documents in libraries and library catalogs, especially from the GBV library network. This includes physical or digital holdings/copies/items as defined in the Functional Requirements for Bibliographic Records (FRBR), the Document Availability Information API (DAIA), as well as general documents as defined in the Bibliographic Ontology (BIBO).

Important relations and properties of items

frbr:part / frbr:partOf relates items to each other by holonymy (part-whole)
daia:exemplar / daia:exemplarOf relates an item to a daia:Document (and by this also bibo:Document and foaf:Document)
daia:unavailableFor / daia:unavailableOf relates an item to a daia:Service which it is not available for
daia:availableFor / daia:availableOf relates an item to a daia:Service which it is available for
daia:heldBy / daia:heldBy relates an item to a daia:Institution that holds the item
dct:description natural language description of the item
dct:spatial (same as dbo:location and schema:location) relates an item to the dct:Location (same as dbo:Place and schema:Place) of a foaf:Document
daia:label A call number, shelf mark or similar label of an item
foaf:page relates an item to a page or document about it
